# نظام المالية ERP - الاتحاد السوري لشركات التأمين

نظام إدارة مالية متكامل للاتحاد السوري لشركات التأمين مع واجهة مستخدم حديثة بتصميم بنفسجي أنيق.

## المميزات

- ✅ تسجيل دخول آمن باسم مستخدم وكلمة مرور
- ✅ لوحة تحكم مالية شاملة
- ✅ دفتر يومية تفصيلي
- ✅ تصميم متجاوب يدعم جميع الأجهزة
- ✅ ألوان بنفسجية أنيقة تطابق هوية الاتحاد
- ✅ مكونات UI حديثة ومُحسّنة

## التثبيت والتشغيل

### المتطلبات
- Node.js 18 أو أحدث
- npm أو pnpm

### خطوات التشغيل

1. تثبيت المكتبات:
```bash
npm install
# أو
pnpm install
```

2. تشغيل المشروع في وضع التطوير:
```bash
npm run dev
# أو
pnpm dev
```

3. فتح المتصفح على:
```
http://localhost:5173
```

## تسجيل الدخول التجريبي

للتجربة، استخدم البيانات التالية:
- **اسم المستخدم**: `admin`
- **كلمة المرور**: `admin123`

## البنية التقنية

- **React 19** - مكتبة واجهة المستخدم
- **TypeScript** - لغة البرمجة
- **Tailwind CSS 4** - إطار التصميم
- **React Router DOM** - التنقل بين الصفحات
- **Axios** - التواصل مع API
- **Vite** - أداة البناء والتطوير

## الصفحات

### 1. صفحة تسجيل الدخول (`/login`)
- تسجيل دخول آمن باسم مستخدم وكلمة مرور
- تصميم عصري مع أيقونات SVG
- رسائل خطأ واضحة

### 2. لوحة التحكم (`/`)
- عرض ملخص الإحصائيات المالية:
  - عدد العمليات
  - إجمالي المبالغ
  - رسوم حكومية
  - رسوم خدمات
  - رسوم الاتحاد/الجهات
- فلاتر تاريخ متقدمة
- تصميم بطاقات جذاب مع أيقونات

### 3. دفتر اليومية (`/ledger`)
- جدول تفصيلي لجميع العمليات المالية
- بحث متقدم حسب:
  - رقم الإيصال
  - رقم البوليصة
  - رقم اللوحة
  - اسم المالك
- فلترة حسب الحالة والتاريخ
- ترقيم صفحات
- تصميم جدول احترافي مع ألوان متناوبة

## مكونات UI المُخصصة

تم إنشاء مكونات UI مُحسّنة تطابق تصميم `newAdmin_sigurta`:

- `Button` - أزرار بتصاميم متعددة (default, outline, destructive, ghost, link)
- `Input` - حقول إدخال مُنسّقة
- `Card` - بطاقات مع header و content و footer
- `Label` - تسميات للحقول

## الألوان

تم اعتماد نظام ألوان بنفسجي أنيق يطابق هوية الاتحاد:

- **Primary**: `rgb(126, 34, 206)` - بنفسجي غامق
- **Border**: `rgb(206, 176, 221)` - بنفسجي فاتح للحدود
- **Background**: تدرج بنفسجي `linear-gradient(135deg, #d8b4f3 0%, #e9d5ff 100%)`

## API Endpoints

يتصل النظام بـ backend عبر:

- `POST /auth/login` - تسجيل الدخول
- `GET /erp/summary` - ملخص الإحصائيات
- `GET /erp/ledger` - دفتر اليومية

## المتغيرات البيئية

يمكن تعيين عنوان API الأساسي:

```env
VITE_API_BASE_URL=http://localhost:3000/api
```
